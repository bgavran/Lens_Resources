# Theory and Applications of Lenses and Optics

This is a list of papers studying the theory and application of lenses and optics, as they're used in category theory and functional programming.
Many of papers are in multiple fields and some of these fields overlap.
Let me know what's missing! (preferably by creating a pull request)
